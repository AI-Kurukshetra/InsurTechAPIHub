# Architecture Decisions

## Framework
Next.js (App Router) chosen for fast development and easy deployment.

## Backend
Supabase used for:
- authentication
- database
- API integration

## Styling
Tailwind CSS for rapid UI development.

## Components
shadcn/ui for accessible UI components.

## Deployment
Vercel used for hosting and CI/CD.

## Design Philosophy
Prioritize:

1. working product
2. simple architecture
3. fast development
